# dmatch

a simple pattern matching implementation to get and organize the results of the dumps


## Installation

```bash
git clone https://github.com/guelfoweb/dmatch.git
cd dmatch
bash dmatch --install
```

## Usage

```bash
dmatch <file> <dump>
```
## Example

```bash
dmatch domain_list.txt dump.txt
dmatch domain_list.txt dump/
```
